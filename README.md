# LottieHud

LottieHUD is a Swift library for showing loading indicator
Which works on navigation controller based iOS applications

## Installation

Use cocoapods to install this custom control in your project. install LottieHUD.

```bash
pod 'LottieHud'
```

## Usage 
```swift
LottieHUD.showHUD()

LottieHUD.hideHUD()
```
